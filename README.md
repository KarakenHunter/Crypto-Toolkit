# Crypto Toolkit

## Overview
Crypto Toolkit is a collection of commonly used cryptographic and encoding utilities implemented in Python. The project is designed to provide simple, modular implementations of hashing algorithms and encoding techniques for learning and experimentation purposes.

This repository focuses on clarity and structure, making it easier to understand how different transformations work internally.

---

## Features

### Hashing Algorithms
- MD5
- SHA-256

### Encoding Techniques
- ROT13
- Base64
